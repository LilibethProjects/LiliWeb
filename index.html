Proyecto realizado con Android Studio, Java y Firebase, el cual está optimizado con funciones reutilizables, clases, objetos, actividades, fragment llevando a cabo un resultado donde contiene al iniciar un registro de usuario con formulario para rellenar usuario, correo y contraseña. Si la contraseña es olvidada te desplega la opción de generar una nueva, al registrarte se desplega un menú el cual tiene las siguientes opciones como tipo de público, buscar, promociones, ubícame, compartir cerrar sesión y eliminar cuenta
